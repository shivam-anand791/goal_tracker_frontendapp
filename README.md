Goal Tracker is a clean and efficient full-stack productivity app designed to help users build habits, track monthly progress, and stay consistent. With a simple dashboard UI and a secure backend, the app offers a structured way to monitor goals and visualize improvement day by day.

🚀 Features
✅ User Authentication

Register and login using JWT

Secure password hashing with bcrypt

Protected API routes for each user

All data isolated per user account

🎯 Habit & Goal Tracking

Add, edit, and delete custom habits

Daily checkboxes for completion

Monthly goal setting per habit

Automatic progress calculation

Month-by-month snapshot storage

📊 Dashboard & Analytics

Habit progress bars

Daily completion summary

Monthly performance overview

Easy navigation between months

💾 Persistent Storage

All data stored in MongoDB Atlas

Auto-sync between frontend and backend

Safe and reliable document structure

🛠️ Tech Stack

Frontend:

HTML, CSS, JavaScript

Responsive dashboard UI

Fetch API for backend communication

Backend:

Node.js + Express

JWT authentication

Bcrypt encryption

Mongoose ODM

Database:

MongoDB Atlas
